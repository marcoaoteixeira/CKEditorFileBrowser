Instruções para upload de arquivos

Para realizar um upload, selecione a pasta de destino no navegador ao lado e, em seguida, clique em "SELECIONAR". Selecione um arquivo para upload e clique em "Enviar arquivos". Você pode selecionar mais de um arquivo por vez, se precisar.

Informação:

A nova pasta será relativa a pasta selecionada anteriormente no navegador de arquivos. Você também pode optar por criar uma estrutura de pastas, a partir de uma pasta relativa, adicionando o caractere "/" para separar os níveis da mesma. Ex.: "<PASTA RELATIVA>/Filho/Neto" Isso irá criar uma pasta chamada "Neto", dentro de uma pasta chamada "Filho", dentro da pasta relativa que foi selecionada.
